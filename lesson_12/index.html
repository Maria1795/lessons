CSS transform : 1) Translate - function

TRANSLATE - Посуває елемент на нове місце. Переміщаючи відносно початковго місця знаходження обєкта вправо та вниз, використовуючи координати X та Y.

transform : translate (x,y);

CSS transform : 2) Scale - function

SCALE - Масштабує елементи,роблячи їх більшими або меншими. 1 - це початковий розмір обєкту.

transform:
scale(0.5);

CSS transform : 3) Rotate - function

ROTATE - Повертає елементи , крутить обєкти.

rotate(n+deg)

CSS transform : 4) Skew - function

SKEW - Деформує (нахиляє елементи)

skew :
(x+deg, y+deg)
